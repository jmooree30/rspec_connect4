## Installation

Open your Terminal/Command Line. Navigate to the directory where your version will live. Type in the following:

```
$ git clone https://github.com/jmooree30/rspec_connect4.git
$ cd rspec_connect4
$ cd connect
$ cd lib 
$ ruby play_game.rb
```
